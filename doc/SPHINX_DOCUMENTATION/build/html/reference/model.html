dassflowmodel#

dassflowmodel is the main class of dassflow2d packagage. It enable interfacing with fortran kernel and python librairies.

dassflowmodel also contains its proper data independently from fortran kernel.

subclasses#

dassflowmodel contains subclasses:

  • to manipulate the mesh: dassflow2d.core.meshing.Meshing

  • to manipulate the configuration file: dassflow2d.core.config.Config

  • to manipulate fortran simulation results: dassflow2d.core.output.Output

  • to manipulate boundary conditions: dassflow2d.core.boundary.Boundary

  • to manipulate parameters: dassflow2d.core.param.Param
